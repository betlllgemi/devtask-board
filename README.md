# DevTask Board

DevTask Board is a Kanban-style task management application developed with React, Vite, and Tailwind CSS.

## Features

- Add new tasks
- List tasks by status
- Edit tasks
- Delete tasks
- Change task status
- Save tasks with localStorage

## Technologies

- React
- Vite
- Tailwind CSS

- components
- pages
- interfaces
